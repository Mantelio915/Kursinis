Instructions:
W,A,S,D - move
Mouse - aim
LMB - shoot

**Known bugs**
-Shooting/ getting shot inside an asteroid results in a crash

**Planned features:**
-Menu/pause states
-Procedural generation
-More enemies
-Powerups
